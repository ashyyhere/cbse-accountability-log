# cbse-accountability-log

This repository documents the ongoing CBSE OSM (Online Scanning and Marking) checking situation, exposing systemic issues in digital evaluation, scan quality, marking reliability, and the transparency of review workflows.

## About the Site
The site collects verified reports and evidence from students, reviewers, and whistleblowers to track problems such as:

- Blurred or incomplete scanned answer sheets
- Incorrect marking and inconsistent evaluation
- Missing pages, bad page order, and lost metadata
- Suspected security gaps in online scanning and result publication systems
- Poor auditability and opaque escalation processes

The goal is to create a traceable archive of cases that helps hold the CBSE OSM checking process accountable and support calls for better oversight.

## Contributors
- Community contributors and witnesses who share verified cases
- Data collectors who document scan and marking issues
- Reviewers and students raising concerns about OSM evaluation

If you want to contribute, open an issue or submit a pull request with evidence-backed reports and anonymized documentation.
